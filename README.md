# Red Bull Inspired Longboarding Website

This project is a high-performance, single-page React website built with Tailwind CSS and Framer Motion, inspired by Red Bull's media house aesthetic. 

It was generated using the following prompt:

## The Prompt

> Act as a Senior Frontend Developer. Create a high-performance, single-page React website (using Tailwind CSS) inspired by Red Bull's media house aesthetic.
> 
> **Key Requirements:**
> * **Hero Section:** Create a video hero component that uses 'Woman_longboarding_mountain.mp4' as a background loop. Overlay it with a bold headline 'THE MOUNTAIN IS CALLING' in uppercase and a primary button that says 'EXPLORE THE LINE'.
> * **Navigation:** Semi-transparent sticky navbar with links for 'Live Events', 'Athletes', 'Gear', and a search icon.
> * **Aesthetic:** Dark mode only. Use `#0A0A0A` for backgrounds, white for body text, and a vibrant orange (`#FF4500`) for highlights and buttons.
> * **Feature Section:** Create a 'Technical Specs' grid with three cards: 'The Deck' (Carbon Fiber), 'The Wheels' (75mm 78A), and 'The Rider' (Safety First).
> * **Animations:** Use Framer Motion or standard CSS transitions for scroll-reveal effects.
> * **Responsiveness:** Ensure the video scales correctly on mobile and the grid collapses to a single column.
> * **Format:** Provide the full code in a single file (HTML/Tailwind) or a clean React component structure.
> 
> **Suggested Content Copy**
> If you are building this out manually, here is the text to make it feel authentic:
> * **Headline:** ASPHALT AT 60MPH
> * **Sub-headline:** Experience the raw intensity of downhill longboarding through the Sierra Nevada range.
> * **The Athlete:** Featuring elite rider Sarah 'The Streak' Miller as she tackles the 12-degree grade.
> * **Call to Action:** [ GET THE GEAR ]

## Tech Stack
- React 19
- Vite
- Tailwind CSS 4
- Framer Motion (for scroll-reveal and micro-interactions)
- Lucide React (for icons)
